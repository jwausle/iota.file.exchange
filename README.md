# FileExchange

Demo implementation to process a encrypted file exchange with email+iota.

# Prerequisite

* java>=8 + kotlin
* mvn>=3.5
* Intellij | eclipse

# Getting started

```$bash
# build
$ git clone ...
$ cd iota.file.exchange
$ mvn clean package

# test
$ java -jar target/iotafe.jar

# edit
$ open Intellij|eclipse
```
